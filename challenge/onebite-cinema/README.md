## 1. 새로운 Next.App 생성하기 (Page Router 버전)

## 2. 라우팅 설정하기
- `/` : 인덱스 페이지
- `/search` : 검색 페이지
- `/movie/[id]` : 영화 상세 페이지

### 1. `/` 인덱스 페이지

### 2. `/search` 검색 페이지

### 3. `/movie/[id]` 영화 상세 페이지
- '[id] 영화 상세페이지' 렌더링
